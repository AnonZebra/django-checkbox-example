This is the home view of the multichoice project site

  1. First of all, you need to run python3 manage.py makemigrations persona and python3 manage.py migrate, if you haven't already.
  2. Once you're up and running, start by adding some characteristics.
  3. Once you've added a few characteristics, follow this link for adding a persona.
  4. On the 'add persona' screen, you should be able to use a multiple checkbox input for choosing characteristics to link to the persona.
  5. When you've added a few personas, you can come back here and you should be able to see them below
